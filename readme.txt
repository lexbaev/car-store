Application which represents a store for car charging sesssion entites.

To run the application go to the directory "/car-store" and 
run script "run.cmd" for Windows or "run" for Linux.

If something is going wrong open a console, go to the "/car-store" folder 
and run the following commands depending on your operatopn system:

Windows:
mvnw package 
java -jar target/car-store-0.0.1-SNAPSHOT.jar

Linux:
./mvnw package 
java -jar target/car-store-0.0.1-SNAPSHOT.jar
